<h2>Requirements</h2>
- vagrant >= 2.2.6
- ansible >= 2.9.6 (also ansible collection 'ansible.windows' is needed)
- python3 >= 3.8.5 (also python modules 'pywinrm' >=0.3.0 is needed)
